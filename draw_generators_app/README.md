# Draw Generators

Приложение для генерации Dive Pool соревнований D2W и D4W.

## Стек
- Python 3.10+
- PySide6
- Pytest

## Запуск
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Тесты
```bash
python -m pytest -q
```

## Тест-кейсы
Сценарии ручной проверки лежат в `tests/TESTCASES.md`.
