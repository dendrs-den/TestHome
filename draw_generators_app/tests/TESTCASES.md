# Manual Test Cases

1. Base generation
- Steps: Set rounds to 5 and keep all checkboxes selected, click `Generate`.
- Expected: 5 rounds are shown with one Snake, one Vertical and one Mixer each.

2. Duplicate logic for small pools
- Steps: Keep only one element in each group, set rounds to 8, click `Generate`.
- Expected: 8 rounds are generated by repeating selected items.

3. Empty group validation
- Steps: Uncheck all elements in Mixers and click `Generate`.
- Expected: Validation warning appears.

4. Reset state
- Steps: Change round count, uncheck some elements, click `Reset`.
- Expected: Round count returns to 10, all checkboxes checked, output cleared.

5. Random reproducibility (automated)
- Steps: Run `python -m pytest -q`.
- Expected: `test_generate_rounds_is_reproducible_with_seed` passes.
