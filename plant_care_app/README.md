# Plant Care App

Приложение для учета ухода за комнатными растениями:
- добавление растений (название и заметка)
- добавление записей о поливе (дата и время)
- просмотр поливов в календаре на выбранный день

## Структура

- `main.py` - точка входа
- `data.json` - хранилище данных
- `models/` - сущности данных
- `services/` - загрузка, сохранение, валидация
- `ui/` - графический интерфейс (tkinter + tkcalendar)
- `tests/` - автоматические тесты

## Требования

- Python 3.10+
- pip

## Установка

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt
```

## Запуск приложения

```bash
python main.py
```

## Запуск тестов

```bash
pytest
```

## Формат данных (`data.json`)

```json
{
  "plants": [
    {"id": 1, "name": "Орхидея", "note": ""}
  ],
  "watering": [
    {"plant_id": 1, "date": "2026-05-15", "time": "18:30"}
  ]
}
```

## Обработка ошибок

- валидация пустого названия растения
- проверка существования растения при добавлении полива
- валидация даты (`YYYY-MM-DD`)
- валидация времени (`HH:MM`)
- обработка некорректного JSON
